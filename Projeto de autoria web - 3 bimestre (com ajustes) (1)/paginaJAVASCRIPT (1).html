POR FIM,IREMOS ESTUDAR JAVASCRIPT:

JavaScript (JS) é uma linguagem de programação popular usada principalmente para adicionar
interatividade e dinamismo a páginas da web, tornando-as mais do que apenas documentos estáticos.

O JavaScript é a linguagem que torna a web interativa, sendo essencial para o desenvolvimento Front-end e, graças ao Node.js, também Back-end.
O que é o NODE.JS?
• É um ambiente de runtime JavaScript de código aberto e multiplataforma
que permite executar código JavaScript fora do navegador, como em servidores.

1-Fundamentos da linguagem

Variáveis e Tipos:
-> Declaração: Use const (para valores que não mudam) e let (para variáveis que serão reatribuídas). Evite var.
-> Tipos Primitivos: String, Number, Boolean, null, undefined, Symbol.
-> Tipos Complexos: Object (que inclui Array e Function).
-> Coerção de Tipos: JS é fracamente tipada. Prefira sempre os operadores de comparação estrita (=== e !==) para evitar conversões automáticas inesperadas (que ocorrem com ==).
Funções:
-> São blocos de código reutilizáveis.
-> Arrow Functions (ES6): Sintaxe concisa (const func = () => {}) muito popular, que possui um comportamento lexical diferente para o contexto (this).

2-Estruturas de Dados e Manipulação

Arrays:
-> Coleções ordenadas.
-> Métodos Essenciais:
• Imutáveis (Funcionais): map() (transforma itens), filter() (seleciona itens), reduce() (combina o array em um único valor).
• Mutáveis: push(), pop(), splice().
Objetivos:
-> Coleções de pares chave-valor, usadas para modelar entidades.
-> ES6+: Desestruturação (Destructuring) (const { prop } = obj;) e Spread Operator (...) para clonar ou combinar objetos e arrays de forma limpa.
