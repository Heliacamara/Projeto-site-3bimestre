CSS

Cascading Style Sheet ou CSS é uma linguagem de programação para organizar melhor o estilo das páginas, ou seja, seu visual, e tem um mecânismo simples para adicionar estilo (fonte, cores, espaçamentos, bordas, etc.)


O CSS é um Web Standard: Um conjunto de normas, detrizes, recomendações produzidos pelo W3C e destinados a orientar desenvolvedores para o uso de práticas que possibilitem a criação de uma Web acessível para todos



REGRA CSS:



A regra do CSS é composta por duas partes: seletor e declaração.



SELETOR {
PROPRIEDADE: VALOR;
}


Seletor: Alvo da regra de estilo;
Declaração: Propriedade e valor;
Propriedade: Define o que será estilizado;
Valor: Define o quanto o como será estilizado.




• A Declaração deve ser colocada obrigatóriamente entre sinais de chave {}.
• O sinal de dois pontos (:) na declaração, separa a propriedade do seu respectivo valor.


exemplo: p {font-size: 20px;}


Neste caso, o código irá deixar a tag p com 20 pixels de tamanho, caso mais uma declaração venha a ser feita, usamos ponto e vírgula (;) para repará-las




Sobre o valor, precisamos ressaltar sobre algo muito importante: quando há mais de duas palavras sem hífem (-), utilizamos as aspas (""), caso o valor contenha mais de duas palavras mas são separadas por hífem (-), não são utilizadas aspas ("")

exemplos:
h1 {font-family: sans-serif;}
h1 {font-family: "Arial Narrow";}





FOLHAS DE ESTILO


Existem 3 formas de declaração de folhas de estilo: Local, incorporada e externa.


• Local: É colocado o atributo "style" dentro da tag do elemento de marcação;
• Incorporada: É colocado a tag "style" dentro do elemento "head";
• Externa: Em um arquivo externo com extensão ".css" e são "linkadas" por meio de um link no elemento "head", sendo o link: "link rel:stylesheet href:'estilo.css'" (sendo "estilo.css" o nome da pasta do css externo).
